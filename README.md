# 🔳 QR Code Generator

QR Code Generator is a Python-based application that creates scannable QR code images from user-provided URLs. The project uses the `qrcode` library to generate QR codes and saves them as PNG image files that can be scanned using smartphones or QR code readers.

This project was created to practice Python libraries, user input handling, image generation, and automation concepts.

- 🔗 Generate QR codes for any website URL

- 📱 Compatible with all QR code scanners

## 🚀 Setup and Installation

### Clone the Repository

```bash
git clone https://github.com/manika7105/QR-Code-Generator.git
```

### Navigate to the Project Folder

```bash
cd QR-Code-Generator
```

### Install Required Packages

```bash
pip install qrcode[pil]
```

### Run the Program

```bash
python qr_code_generator.py
```

---

## 🧾 How to Use

1. Run the Python script.
2. Enter the URL you want to convert into a QR code.
3. Enter a file name for the generated image.
4. The QR code will be created and saved as a PNG file.
5. Scan the generated QR code using your smartphone.

### Example

```text
========== QR Code Generator ==========

Enter a URL: https://github.com/manika7105

Enter output file name (without .png): github_qr

QR Code generated successfully!
Saved as: github_qr.png
```
