# 🔳 YouTube QR Code Generator using Python

This is a **Python-based QR Code Generator** that creates a scannable QR code for a given YouTube video link. The QR code is generated using the **`qrcode`** library and saved as an image file.


## ✨ Features

- 🔗 Converts any URL into a QR code
- 🖼️ Saves the QR code as a PNG image
- ⚙️ Customizable size, color, and border
- ⚡ Quick and lightweight script


## 🛠️ Technologies Used

- 🐍 **Python 3**
- 📦 **qrcode** – Python library for generating QR codes
- 🖼️ **Pillow** – Imaging library used internally by `qrcode`


## 🚀 How to Run

1. **Install Required Packages:**  
   If not already installed, run:
   ```bash
   pip install qrcode[pil]
   ```
2. **Run the Script:**
   ```bash
   python qr_generator.py
   ```
3. The QR code will be saved as:
   ```bash
   youtube_qr.png
   ```


## 🧾 Output  

📸 A QR code image (```youtube_qr.png```) that, when scanned, redirects to the YouTube video.
