# 🔳 QR Code Generator

QR Code Generator is a Python-based application that creates scannable QR code images from URLs. The project uses the `qrcode` library to generate QR codes and saves them as image files that can be scanned using smartphones or QR code readers.

This project was created to practice Python libraries, image generation, and automation concepts.

---

## ✨ Features

* 🔗 Convert URLs into QR codes
* 🖼️ Save QR codes as PNG images
* ⚡ Lightweight and easy to use
* 📱 Compatible with smartphone QR scanners
* ⚙️ Customizable size and border settings

---

## 🛠️ Technologies Used

* 🐍 Python 3
* 📦 qrcode Library
* 🖼️ Pillow (PIL)

---

## 🚀 Setup and Installation

### Clone the Repository

```bash
git clone https://github.com/manika7105/QR-Code-Generator.git
```

### Navigate to the Project Folder

```bash
cd QR-Code-Generator
```

### Install Dependencies

```bash
pip install qrcode[pil]
```

### Run the Program

```bash
python qr_code_generator.py
```

---

## 🧾 How It Works

1. A URL is provided in the script.
2. The application generates a QR code using the `qrcode` library.
3. The QR code is converted into an image.
4. The image is saved as a PNG file.
5. Users can scan the QR code to open the associated link.
